**EmptyState** — empty screens as an invitation to act, never a mood. Framed glyph, one-line copy, a single primary action.

```jsx
<EmptyState
  icon={<PackageIcon/>}
  title="No modules connected yet"
  actions={<Button>Connect module</Button>}
>
  Run a composer require, or paste a license key to activate your first module.
</EmptyState>
```

Keep the body to one concrete line and one primary action.
